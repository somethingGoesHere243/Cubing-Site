Notation

Each move of the Rubik's cube can be notated by just two charaters, the first to represent the face/slice being rotate, and the second to represent the angle which it is being rotated by.

The second symbol can take one of two values:

  • If no second symbol is present the face should be rotated 90 degrees clockwise (as if you were facing that side).
  • If the second symbol is a ' the face should be rotated 90 degrees anticlockwise.
  • If the second symbol is a 2 the face should be rotated 180 degrees.
